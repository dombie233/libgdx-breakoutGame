🧱 Breakout Game – libGDX

A classic Breakout-style arcade game built with libGDX in Java.
Control the paddle, bounce the ball, and destroy all bricks.

🎮 Gameplay
- Move the paddle left and right
- Bounce the ball to destroy bricks

🛠️ Tech Stack
- Java
- libGDX
- Gradle

🧩 Features

 - [x] Paddle movement

 - [x] Ball physics and wall bouncing

 - [x] Brick collision detection

 - [x] Sound effect when the ball destroys a brick

 - [x] Desktop support (LWJGL3)

 - [x] Web support (GWT)
